# 🚗 Car Rental System (Secure Coding - Java)

A secure car rental management system developed using Java, focusing on applying secure coding practices and role-based access control.

---

## 📌 Project Description

This project is a car rental system built as part of a secure coding course. The system allows different types of users to interact with the platform, including administrators, maintenance staff, employees, and customers.

The main goal of this project is to demonstrate secure software development practices while managing a real-world scenario (car rental service).

---

## 🔐 Security Features

* 🛡️ Input validation to prevent invalid or malicious data
* 👥 Role-based access control (RBAC)
* 🚫 Protection against common vulnerabilities (e.g., injection attacks)
* 🔒 Secure handling of user data

---

## 🚀 System Features

* 🚗 Car rental and booking system
* 👤 User registration and login
* 🧑‍💼 Admin dashboard for managing users and cars
* 🔧 Maintenance staff can manage car servicing
* 🗂️ Employees can handle operations and requests

---

## 👥 User Roles

* **Admin**: Full control over the system
* **Maintenance Staff**: Manage car maintenance and status
* **Employees**: Handle bookings and customer requests
* **Customers**: Rent cars and view available options

---

## 🛠️ Technologies Used

* Java
* OOP (Object-Oriented Programming)
* Secure Coding Practices

---

## ⚙️ How to Run

1. Open the project in your Java IDE (IntelliJ / Eclipse)
2. Compile and run the project
